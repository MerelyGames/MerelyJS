MerelyJS API

new MerelyJS()
API:  new MerelyJS();
Справка:  Создает копию игрового движка.
Пример:  merelyjs = new MerelyJS();
Возвращает:  (object) игровой движок с другими объектами.

new MerelyJS().Display(width, height, bgColor);
API:  Display(width, height, bgColor);
Справка:  Создает canvas с параметрами: clearDisplay() - очищает весь холст, width (длина), height (высота), bg (фон холста).
Пример:  merelyjs.Display(width, height, bgColor);
Возвращает:  (object) объект дисплея.

new MerelyJS().GameLoop(function, fps);
API:  GameLoop(function, fps);
Справка:  Запускает новый игровой цикл function со скоростью fps.
Пример:  merelyjs.GameLoop(function, fps);
Возвращает:  (object) игровой движок с другими объектами.

new MerelyJS().CreateGameObject(x, y, width, height, src, alpha, objData);
API:  CreateGameObject(x, y, width, height, src, alpha, objData);
Справка:  Создает игровой объект, где src - путь к файлу изображения, alpha - прозрачность, objData - доп. свои данные для объкта.
Пример:  merelyjs.CreateGameObject(x, y, width, height, src, alpha, objData);
Возвращает:  (object) объект со свойствами и значениями: clearDisplay() - очищает весь холст, width (длина), height (высота), bg (фон холста).
